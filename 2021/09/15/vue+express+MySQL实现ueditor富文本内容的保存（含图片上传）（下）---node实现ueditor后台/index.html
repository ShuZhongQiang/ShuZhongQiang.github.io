Installation

一、下载 UEditor

下载最新编译的 UEditor。官网目前最新的版本是1.4.3.3，存在诸多 BUG，例如
Issue1，且官方不再积极维护。为了世界的和平，针对一些常见 BUG，我进行了修复，并把编译好的文件放在了本仓库的
assets/downloads 目录下，你可以放心下载，当然你也可以自己 clone 官方源码并编译。


将下载的压缩包解压并重命名为 UEditor（只需要选择一个你需要的版本,比如 utf8-php）,放入你项目的 static 目录下。

如果你使用的是 vue-cli 3.x，可以把 UEditor 文件夹放入项目的 public 目录下。
